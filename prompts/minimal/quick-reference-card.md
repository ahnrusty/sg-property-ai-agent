|---------|-----|-----|------|

## LOAN RULES

- **TDSR 55%**: total debt incl new mortgage at stress rate, ÷ gross monthly income
- **MSR 30%**: HDB / EC from developer, mortgage only, stress rate
- **Stress rate**: 4% bank residential / 3% HDB loan / 5% non-residential
- **LTV**: 75% first / 45% second / 35% third+
- **Min cash**: 5% first / 25% second+
- **Max tenure**: HDB 25yr (age 65 cap); private 30yr (age 75 cap)

Variable income haircut: 30% (use 70%).

---

## CPF

- OA usable: down payment, mortgage, BSD, new-launch ABSD, legal
- NOT usable: resale ABSD, renovation, maintenance, property tax, agent commission (buyer side)
- Lease cover **youngest buyer to age 95** for full VL; pro-rated below; ZERO if cover < age 65
- Accrued interest: 2.5% compounded; return on sale
- Withdrawal Limit: 120% of VL (BRS set aside if 55+)
- At 55+: refund tops up RA to FRS ($220,400 for 2026; $228,200 for 2027)

## HDB FRAMEWORK (Oct 2024+)

| Tier | MOP | Clawback | Resale |
|------|-----|----------|--------|
| Standard | 5yr | None | SC + SPR |

## UPGRADE / DOWNGRADE QUICK PATHS

| Path | Best move |
|------|-----------|
| HDB → Condo (SC couple) | Buy first + sell HDB in 6mo (ABSD refunded) |
| HDB → Condo (single SC) | Sell first ($0 ABSD) |
| HDB → Landed | SC only; same logic as HDB → Condo |
| Condo → Condo (keep current) | Decoupling (saves $300k+) |
| Condo → Landed (keep condo) | Decoupling, spouse A buys landed as first property |
| Landed → Condo (under 55) | Sell first |
| Landed → Condo (55+) | Sell first; CPF refund boosts RA → higher CPF LIFE |
| Condo / Landed → HDB (under 55) | Wait 15 months from disposal, then buy HDB resale |
| Condo / Landed → HDB (both 55+, 4-room or smaller) | NO wait; senior exemption |
| HDB → HDB | No ABSD; resale levy if previously subsidised |

## HARD DEADLINES

- **OTP exercise**: 14 days (private) / 21 days (HDB resale)
- **Stamp duty payment**: 14 days from OTP (30 if overseas)
- **6-month ABSD remission**: from new property legal completion (resale) or TOP/CSC (new launch). **NO extensions**.
- **15-month wait-out**: from private property disposal completion. Senior exemption: both 55+, 4-room or smaller.
- **30-month wait** for ex-private owners applying for new BTO

---

## FOREIGNER / SPR LANDED

- Foreigner: mainland landed NO (LDAU approval rare); Sentosa Cove only, one unit, own-occupation
- SPR: mainland landed NO without LDAU approval
- Strata landed in condo development: classified as non-landed; foreigners can buy

---

## RENTAL (TENANT)

- Lease 12-24mo; deposit 1mo per yr lease; advance 1mo
- Stamp duty: 0.4% of total rent (≤ 4yr lease); tenant pays typically
- Diplomatic clause: 12mo minimum, 2mo notice, foreigners only
- Minor repair cap: tenant pays up to $150-$200; landlord above
- HDB whole-flat sublet: needs HDB approval + MOP fulfilled

---

## CRITICAL "DO NOT" LIST

- Don't quote a price you didn't get from URA Realis / HDB Resale Stats / actual listing
- Don't promise ABSD refund without confirming SC married couple + 6-month timing
- Don't recommend decoupling without checking sole-name loan qualification
- Don't ignore 15-month wait-out for condo → HDB downgraders under 55
- Don't claim foreigner can buy mainland landed
- Don't compute stamp duties or affordability mentally; use the calculator
- Don't sign anything without engaging a CEA agent and conveyancing lawyer
