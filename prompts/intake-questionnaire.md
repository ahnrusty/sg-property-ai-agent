# Intake Questionnaire

Use this at the start of any new engagement. Fill what is known, ask the rest. Keep it tight; do not interrogate.

## 1. Intent

- [ ] Buy own-stay
- [ ] Buy investment
- [ ] Sell
- [ ] Rent (tenant)
- [ ] Let out (landlord)
- [ ] Refinance research
- [ ] General market research

## 2. Buyer or seller profile

- Residency: Singapore Citizen / Singapore PR / Foreigner
- Household: single / married / family with children / extended
- Spouse residency (if married): SC / SPR / Foreigner
- Number of residential properties already owned (Singapore + overseas if relevant)
- For HDB: any prior subsidised flat? Any prior CPF Housing Grant?

## 3. Budget and finances

- Target max price (SGD)
- Cash available for down payment, stamp duty, legal, renovation
- CPF OA balance usable
- Loan IPA: not started / in progress / approved
- IPA max loan quantum and rate package, if approved
- Gross monthly household income (for affordability check)
- Existing monthly debt obligations (car, credit cards, other property loans)

## 4. Property scope

- Type: HDB / EC / Private condo / Landed / Open
- For HDB: flat type (2-room Flexi, 3-room, 4-room, 5-room, Executive, 3Gen)
- For EC: launch or resale
- For condo: new launch only / resale only / both
- For landed: terrace / semi-D / detached / GCB / strata landed
- Tenure preference: freehold only / 999-year / 99-year acceptable
- Age preference: brand new / under 10 yrs / under 20 yrs / no preference

## 5. Location

- Preferred districts or planning areas
- Preferred MRT lines and walking radius (e.g. ≤ 10 min walk)
- School priorities (name the school and tier)
- Commute anchors (workplace address, max travel time)
- Hard nos (specific areas to avoid and why)

## 6. Specifications

- Bedrooms (min / ideal)
- Bathrooms (min / ideal)
- Floor area (sqft or sqm; min / ideal)
- Parking (mandatory / nice to have / not needed)
- Pet ownership (specify type and size if condo, for MCST rules)
- Storey preference (low / mid / high; specific facing if known)
- Renovation tolerance (turnkey / light reno / full reno OK)

## 7. Timeline

- Target completion / move-in date
- Earliest possible
- Latest acceptable
- Any anchoring events (school start, lease expiry on current home, sale of existing property)

## 8. Other constraints

- Foreigner ABSD constraints (60% on any private)
- 15-month wait-out (private owner buying HDB resale)
- HDB MOP if currently owning
- CPF accrued interest concerns if selling current property
- Decoupling considerations for second property
- Any pending family changes (marriage, divorce, citizenship, baby)

## 9. What success looks like

In one or two lines, what does a great outcome look like for this engagement? (E.g. "Shortlist of 5 OCR 4-bedroom condos under $2.4M near MRT, ready to book viewings within 2 weeks.")
